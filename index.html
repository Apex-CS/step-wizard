STEP 1 - EXAMPLE The step wizard requires a div with the attributes set to class="setup-content" id="step-"(step number)".
It will detect if the inputs inside the div are required and disable or enable the Next button.
NOTE: uses JQuery Validator to display the error messages on each invalid input.

STEP 2 NO BOOTSTRAP - BASIC CONFIG

The Next and Back need the class class="nextBtn" and class="backBtn" respectively.


STEP 3 - EXTRA LIBRARIES This example has Bootstrap Datepicker and Bootstrap Selectpicker integrated to work with the step wizard validator, but are not required
